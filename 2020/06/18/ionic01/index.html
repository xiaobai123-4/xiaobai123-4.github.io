ionic

官网: https://ionicframework.com/

官方教程: https://ionicframework.com/getting-started

组件文档: https://ionicframework.com/docs/components/

介绍：ionic是一款第三方的混编库, 用于使用前端语言来制作手机端项目iOS, Android

混编: 混合编译 (Hybrid App)

  • 15年之前, 开发android 和 iOS 必须使用原生的语言: java 和 Objective-C, Swift, 这些语言开发的原生项目包打包之后才可以运行在andriod和苹果手机上

    15年之前, 教iOS开发

  • 手机端整体和前端非常像, 很多前端程序员也想开发手机端软件 但是 又不愿意学 新的语言

    • Facebook 开发了最初的ReactNative, 可以利用js来书写代码 然后经过编译之后, 变为手机端的软件, 在手机上执行
    • 写一次 就可以分别编译成 android版本 和 iOS版本同时执行!
    • 小公司节约成本, 普遍选择混编方式进行开发手机端项目

  • 缺点: 混编项目的运行效率 已经能够实现的功能 没有原生的多, 没原生的强

    • 小型项目, 对客户体验要求不高 小公司 适合混编
    • 大型项目, 对用户体验要求高, 适合原生开发

混编开发目前最主流的两款框架

  • ionic
    • 基于 vue, angular,react都可以做开发
    • 特色: 更趋近与网页方式呈现页面
  • ReactNative
    • 基于React进行开发
    • 特色: 更趋近原生的App, 效率更高, 能够调用更加丰富的原生API

ionic开发

需要提前安装脚手架:

确认为中国镜像

npm config get registry

中国镜像 带有 taobao 字样

设置中国镜像

npm config set registry https://registry.npm.taobao.org/

然后安装即可:

npm i -g @ionic/cli
  • @ionic/cli@6.2.1

使用脚手架, 在指定目录下生成项目包

ionic start ionic-app blank

含义: 生成项目名:ionic-app   类型:空白项目

过程中: 一路回车即可

启动项目:

在项目所在目录下, ionic-app 下

ionic serve

简写:
ionic s

如果环境没有安装成功的同学, 直接使用老师给的压缩包, 则命令行要加 npx 前缀
npx ionic s

认识ionic脚手架

  • 访问 localhost:8100 查看项目
  • 实际上触发的是 index.html 文件, 其中含有组件``
  • app-root 就是 app.component.ts组件

badge组件

<!-- 认识一些基础的ionic标签 -->
<!-- 注意: 苹果设备 和 android设备的页面效果会有所差异! -->

<!-- 安装了插件之后, 所有ionic提示 都通过 i- 触发 -->

<!-- 对比HTML来说: ion-app 就是 html 标签 -->
<ion-app>
  <!-- i-header 生成 -->
  <ion-header>
    <ion-toolbar>
      <ion-title>Header</ion-title>
    </ion-toolbar>
  </ion-header>

  <!-- 类似于body, 用于存放内容代码 -->
  <ion-content>
    <!-- 徽章组件 badge -->
    <!-- https://ionicframework.com/docs/api/badge -->

    <!-- 默认蓝底 -->
    <ion-badge>99</ion-badge>

    <!-- 颜色:  -->
    <!-- color 是 ion-badge的一个属性, 用于设定组件的背景色, 具有固定的值 -->
    <ion-badge color="primary">浅蓝色</ion-badge>
    <ion-badge color="secondary">青色</ion-badge>
    <ion-badge color="tertiary">蓝色</ion-badge>
    <ion-badge color="success">绿色</ion-badge>
    <ion-badge color="warning">黄色</ion-badge>
    <ion-badge color="danger">红色</ion-badge>
    <ion-badge color="light">浅灰</ion-badge>
    <ion-badge color="medium">灰色</ion-badge>
    <ion-badge color="dark">黑色</ion-badge>
  </ion-content>
</ion-app>

输入框组件

https://ionicframework.com/docs/api/input

<!-- 输入框组件 -->
<!-- https://ionicframework.com/docs/api/input -->
<ion-app>
  <ion-header>
    <ion-toolbar>
      <ion-title>输入框组件</ion-title>
    </ion-toolbar>
  </ion-header>

  <ion-content>
    <!-- autofocus 自动焦点 -->
    <ion-input autofocus type="text" placeholder="请输入姓名"></ion-input>
    <!-- clearInput 输入框右侧出现x 点击之后清空内容 -->
    <ion-input clearInput type="password" placeholder="请输入密码"></ion-input>

    <ion-input type="password" placeholder="Awesome Input"></ion-input>
  </ion-content>
</ion-app>

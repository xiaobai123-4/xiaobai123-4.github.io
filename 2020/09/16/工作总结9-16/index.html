设备补货模块renderAddDialog

  1. 下拉框的理解

<Input type="select"
    // 方便insert操作数据库,name对应插入字段
    name="title"
    // input框提示信息
    message={this._lang('product_name')}
    // 元素待选择的选项
    option={product_info}
    placeholder='请选择'
    onValueChange={(e) => {
        // value
        // console.log(e)
        if(this.state.product_info) {
            // let index=product_info.findIndex(item=> item.value===e)
            // 用findIndex获取请求的整个数组中所输入条形码对应的一条数据
            product_info = product_info[findIndex(product_info, {"value": e})]
            // console.log(product_info)
            // 解决报错undefind和null
            if(!empty(this.state.product_info))
            this.setState({
                product_price: product_info.price,
                product_img: product_info.img,
                product_id: product_info.product_id
            })
        }
    }}
/>

  1. onchange的用法

<Input type="text"
        name="barcode"
        message={this._lang('barcode')}
        placeholder={"请输入条形码"}
        onChange={(e)=>{
            // 设置一个定时器
            clearTimeout(this.state.item_timer);
            this.setState({code:e.target.value});
            this.state.item_timer=setTimeout(()=>{
                // 发送请求传入code值
                this.http1.send(window.SERVER_URL+'/chenxuan/supply/get_goods_info',{code:this.state.code})
                    .success((model)=>{
                        console.log(model)
                        this.setState({product_info:model});
                    })
            },800)
        }}
/>

  1. 后台数据请求的方法

@PropAction({auth: false})
async get_goods_info({}, {code}) {
    if (code) {
        let model = await this.db('jh_machine_product')
            .whereRaw(`barcode like ?`, [`%${code}%`])
            .select('name as title', 'id as value', 'img as img', 'price as price')
        if (empty(model)) {
            this.callback(null, {toast: "没有找到对应商品"});
        } else {
            this.callback(model, {toast: "已展示所有商品"});
        }
    } else {
        this.callback(false, {toast: "请重新输入条形码"});
    }
}

  1. 其他常见用法

    • empty的用法
      当我们需要防止一个值为空的时候就执行后面代码导致报错undefind和null时候可以使用if(!empty(project_name))
    • 模糊搜索
      let model = await this.db('jh_machine_product')
                      .whereRaw(`barcode like ?`, [`%${code}%`])
                      .select('name as title', 'id as value', 'img as img', 'price as price')
    • 添加只读权限
      readOnly={true}
    • model的用法
      使得所需字段与获取对象中的字段一一对应
    • 必填字段的两种写法
      ①在from中定义某些字段是必填字段可以直接在from标签中定义rules={xx:'required'}
      ②在编辑或者添加功能中哪些字段是不可或缺的就在指定文件夹demo/assets/app/admin/validate/admin.json中方法下的添加该模块-该方法-该字段

//获取条形码
@PropAction({auth: false})
async get_goods_info({}, {code}) {
    if (code) {
        let model = await this.db('jh_machine_product')
        .whereRaw(`barcode like ?`, [`%${code}%`])
        .select('name as title', 'id as value', 'price as price')
        if (empty(model)) {
            this.callback(null, {toast: "没有找到对应商品"});
        } else {
            this.callback(model, {toast: "已展示所有商品"});
        }
    } else {
        this.callback(false, {toast: "请重新输入条形码"});
    }
}

第一次只能赋值无法获取到值的信息

//重新定义一个模块用来展示machine_id,machine_id是通过区域和线路来获取的,所以还要传入edit中的area_id和line_id
renderMachine(edit){
	let {machines} = this.state;
    // debugger;
    if(!empty(machines)){
        return(
            <Input
                type="checkgroup"
                name="machine_id"
                model={edit}
                option={machines}
                message={this._lang('machine_name')}
                onValueChange={() => {
                    this.forceUpdate()
                }}
                placeholder="all"
            />
        )
    }
}

在编辑的时候,也是需要获取area_id和line_id,在获取二级的ID时,在添加中时原本就可以为空,而在编辑时,需要原本就有值,因此需要根据area_id分组,在弹窗之前就要传入area_id,在原本的模块中添加{this.renderMachine(edit)}

:checked实现单选

/*css*/
    .radio_tab:checked~.info{
      display: block;
      z-index: 999;
    }

    .radio_tab{
      display: none;
    }
    .book .info {
      width: 130px;
      height: 190px;
      text-align: center;
      line-height: 200px;
      background-color: #ba985d;
      box-shadow: -3px 4px 2px #888888;
      position: absolute;
      top: 80px;
      left: 20px;
      z-index: 999;
      /* z-index: -1; */
      display: none;
    }
/*css*/


<div class="book">
	<input type="radio" name="radio_tab" id="tab_B_1" class="radio_tab">
	<label for="tab_B_1">
		<span>書1</span>
	</label>
	<div class="info">B-1詳情</div>
	<input type="radio" name="radio_tab" id="tab_B_2" class="radio_tab">
	<label for="tab_B_2">
		<span>書2</span>
	</label>
	<div class="info">B-2詳情</div>
</div>

type为checkbox时为复选框 ,改变样式是通过改变label的样式实现样式多样化

兄弟选择器的使用

‘+’选择器则表示某元素后相邻的兄弟元素，也就是紧挨着的，是单个的。而‘~’选择器则表示某元素后所有同级的指定元素，强调所有的。

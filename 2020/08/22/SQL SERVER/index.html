创建数据库

常见数据库操作,最常用的是对表的一些查询方法

#丢弃数据库bookstore，如果存在的话
DROP DATABASE IF EXISTS bookstore;
#创建数据库bookstore
CREATE DATABASE bookstore CHARSET=UTF8;
#进入该数据库
USE bookstore;
#创建保存图书数据的表
CREATE TABLE store(
  bid INT,
  title VARCHAR(16),
  price INT
);
#插入数据
INSERT INTO store VALUES('11','《JAVA程序设计》','78');
INSERT INTO store VALUES('12','《JavaScript高级程序设计》','99');
INSERT INTO store VALUES('13','《Speaking JavaScript》','108');
INSERT INTO store VALUES('14','《1234》','18');
INSERT INTO store VALUES('15','《1fdfd》','65');
#简写
INSERT INTO store VALUES(21,'《SQL Server数据库》',55),(22,'《c++技术编程设计》',66),(23,'《c语言程序设计》',65);
#查询数据
SELECT * FROM store;
#删除数据
DELETE FROM store WHERE bid='14';
#修改数据
UPDATE store set title='《计算机网络基础》',price=55 WHERE bid='15';

复杂查询

聚合查询/分组查询

示例: 查询所有员工的数量
SELECT COUNT(ename) FROM emp;
练习: 分别使用部门编号和员工编号获取数量
SELECT COUNT(deptId) FROM emp;
SELECT COUNT(eid) FROM emp; #推荐写法
练习: 查询出所有男员工的数量
SELECT COUNT(*) FROM emp WHERE sex=1;

聚合函数
函数是一个功能体，提供若干个数据，产出特定结果——饺子机
COUNT()/SUM()/AVG()/MAX()/MIN()

练习: 查询所有员工的工资总和
SELECT SUM(salary) FROM emp;
练习: 查询出男员工的平均工资
SELECT AVG(salary) FROM emp WHERE sex=1;
SELECT SUM(salary)/COUNT(salary) FROM emp WHERE sex=1;
练习: 查询出工资最高的员工
SELECT MAX(salary) FROM emp;
练习: 查询出年龄最大的员工
SELECT MIN(birthday) FROM emp;

分组查询: 只能查询分组条件和聚合函数

示例: 查询出每个部门员工的最高工资，最低工资，平均工资
SELECT deptId,MAX(salary),MIN(salary),AVG(salary) FROM emp GROUP BY deptId;
练习: 分组查询出男女员工的数量，最高工资和最低工资
SELECT sex,COUNT(eid),MAX(salary),MIN(salary) FROM emp GROUP BY sex;
示例: 查询日期中的年份
SELECT YEAR('2019-11-6');
示例: 查询日期中的月份
SELECT MONTH('2019-11-6');
练习: 查询出1991年出生的员工有哪些
SELECT * FROM emp WHERE YEAR(birthday)=1991;
练习: 查询出3月份生日的员工有哪些
SELECT * FROM emp WHERE MONTH(birthday)=3;
(2)子查询
示例: 查询出研发部的员工有哪些
步骤1: 查询出研发部的部门编号
SELECT did FROM dept WHERE dname='研发部';
步骤2: 根据所得到的编号查询对应的员工
SELECT * FROM emp WHERE deptId=10;
综合:
SELECT * FROM emp WHERE deptId=(SELECT did FROM dept WHERE dname='研发部');
练习: 查询出比tom工资高的员工有哪些
步骤1:查询出tom的工资是多少
SELECT salary FROM emp WHERE ename='tom';
步骤2:根据tom工资查询员工
SELECT * FROM emp WHERE salary>6000;
综合：
SELECT * FROM emp WHERE salary>( SELECT salary FROM emp WHERE ename='tom');
练习: 查询出和tom同一年出生的员工有哪些
步骤1: 查询出tom出生的年份
SELECT YEAR(birthday) FROM emp WHERE ename='tom';
步骤2: 查询年份对应的员工有哪些
SELECT * FROM emp WHERE YEAR(birthday)=1990;
综合:
SELECT * FROM emp WHERE YEAR(birthday)=( SELECT YEAR(birthday) FROM emp WHERE ename='tom') AND ename!='tom';

多表查询

查询出所有的员工姓名及其部门名称
SELECT ename,dname FROM emp,dept;
错误: 笛卡尔积
解决方法: 添加查询条件，两个表之间的关联(外键和主键)
SELECT ename,dname FROM emp,dept WHERE deptId=did;
上述语法无法查询出没有部门的员工，无法查询出没有员工的部门
(1)内连接——和之前查询结果一致
SELECT ename,dname FROM emp INNER JOIN dept ON deptId=did;
(2)左外连接——显示左侧表中所有的记录
SELECT ename,dname FROM emp LEFT OUTER JOIN dept ON deptId=did;
先写哪个表哪个表就是左侧的，OUTER关键字可以省略
(3)右外连接——显示右侧表中所有记录
SELECT ename,dname FROM emp RIGHT OUTER JOIN dept ON deptId=did;
后写哪个表哪个表就是右侧，OUTER关键字可以省略
(4)全连接
显示左侧和右侧表中所有记录——FULL JOIN
mysql不支持全连接
UNION 合并相同的记录
UNION ALL 不合并相同的记录
把左外连接的结果和右外连接的结果合并起来
(SELECT ename,dname FROM emp LEFT OUTER JOIN dept ON deptId=did) UNION (SELECT ename,dname FROM emp RIGHT OUTER JOIN dept ON deptId=did);

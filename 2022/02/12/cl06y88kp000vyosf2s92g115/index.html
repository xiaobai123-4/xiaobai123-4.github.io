vuex

关键字： 保证状态以一种可预测的方式发生变化 ，集中管理状态，单向数据流

state 存放数据类似于实例中data
getters 计算属性类似于实例中computed
mutations 方法类似于实例中methods
actions 异步方法以及commit提交mutations中的方法
modules 模块化，将 store 分割成模块（module）。每个模块拥有自己的 state、mutation、action、getter、甚至是嵌套子模块——从上至下进行同样方式的分割
strict 严格模式下直接修改state里面的值会报错，

mutations和actions

mutations:必须是同步函数，可以直接修改state数据
actions:可以包含任何异步操作，actions也可以直接修改state数据，但是建议通过commit提交mutations去修改state数据，actions调试方便
strict:true 严格模式下直接修改state里面的值会报错，

主要是为了状态能够很好的跟踪，开发者工具更好的处理状态变化，更好的调试体验

computed、watch，methods

computed: 数据有缓存，进入组件时会执行，当值发生变化时会执行
watch: 监听数据当值发生变化时可以获取原始值和变更的值，例如监听路由的变化，只有当值发生变化时会执行
methods： template改变时执行

由三类构成：访问器属性+观察者模式+虚拟DOM树

Object.defineProperty中get()负责把受保护的属性值传给外部，set()负责修改属性值

let obj = {}
let num = 12
// 在对象中添加一个属性与数据描述符的示例
Object.defineProperty(obj, "num", {
    get() {
        return num
    },
    set(newVal) {
        console.log("===")
        num = newVal
    },
    // writable: true,
    // enumerable: true,
    // configurable: true
});

观察者模式(observer)：一个变量被修改时，可以自动通知所关注这个变量的对象，让他们自动重新获得这个变量的新值

虚拟DOM树：仅保存可能发生变化的少量元素的精简版DOM树
优点：小，遍历快；只更新受影响的元素；封装了DOM操作，无需重复编码

原型链

每个对象创建的时候都有自己的隐式原型和构造函数

let obj={}
console.log(obj.constructor)  //ƒ Object() { [native code] }
console.log(obj.__proto__)  //ƒ {constructor: ƒ,__proto__: (...),…}
console.log(obj.__proto__ == Object.prototype) //true
console.log(Object.prototype.__proto__) //null
obj的constructor是Object那么obj的隐式原型__proto__就等于Object的显示原型prototype
查找对象的属性时先在对象本身找然后在对象的原型找然后去对象的原型的原型上找，直到找到最顶端null

继承 （people instanceof Home)

//原型继承

//①class里面的extends
class Home {
    constructor(money) {
        this.money = money
    }
    get money() {
        return this._money;
    }
    set money(value) {
        this._money = value;
    }
    talk() {
        console.log(`我是${this.username},我有￥${this.money}`)
    }
}
class People extends Home {
    constructor(money) {
        super(money)
        this.username = "People"
    }
}

// ②构造函数
function Home(money) {
    this.money = money;
    this.talk = function () {
        console.log(`我是${this.username},我有￥${this.money}`)
    }
}
function People(money) {
    this.username="people"
    Home.call(this,money)
}
let people = new People(12)
people.talk()

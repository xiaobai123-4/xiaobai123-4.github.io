路由

根组件自动挂载组件,动态往根组件里面加载组件

1.安装VueRouter

npm install vue-router --save

2.新建路由文件router/router.js并引入vue-router,或者直接在main.js中直接引入

import VueRouter from 'vue-router'
Vue.use(VueRouter)

前提:必须引入Vue才能使用 import Vue from 'vue'

3.创建组件

import *** from '../components/***.vue'

4.配置路由

const routes=[
    {path:'/**',compontent:***},
     ...
]

5.实例化VueRouter

const router=new VueRouter({
  mode:'history',
  routes:routes,
})

6.挂载路由

//在Vue实例中挂载路由
new Vue({
  router,
  render: h => h(App),
}).$mount('#app')

7.在根组件模板里面放入<router-view></router-view>子组件会动态加载在这里<router-link to="/**">跳转测试</router-link>

8.动态路由

//main.js中配置动态路由
routes:[
	//动态参数 :##
	{path:"/user/:id",component:User}
]
//父组件路由跳转时添加参数
<router-link :to="'/user/'+xx">跳转测试{{xx}}</router-link>
//子组件获取动态路由的值
this.$route.params.id

9.get传值

//main.js中配置动态路由
routes:[
	//动态参数 :##
	{path:"/user",component:User}
]
//父组件路由跳转时添加参数
<router-link :to="'/user?id='+XX">跳转测试{{xx}}</router-link>
//子组件通过get获取传值
this.$route.query.id

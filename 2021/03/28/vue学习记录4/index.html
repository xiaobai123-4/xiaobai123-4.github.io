vue父子组件传参

父组件传值给子组件

1. 父组件调用子组件时通过双向绑定的方式传递父组件定义的数据

<v-header :title="title"></v-header>

2. 子组件通过props接收父组件传递过来的数据

//①数组形式
props:['***']
//②希望每个 prop 都有指定的值类型。这时，你可以以对象形式列出 prop，这些 //property 的名称和值分别是 prop 各自的名称和类型：
props: {
  title: String,
  likes: Number,
  isPublished: Boolean,
  commentIds: Array,
  author: Object,
  callback: Function,
  contactsPromise: Promise // or any other constructor
}
//③为 props 中的值提供一个带有验证需求的对象，而不是一个字符串数组。
props: {
    // 基础的类型检查 (`null` 和 `undefined` 会通过任何类型验证)
    propA: Number,
    // 多个可能的类型
    propB: [String, Number],
    // 必填的字符串
    propC: {
      type: String,
      required: true
    },
    // 带有默认值的数字
    propD: {
      type: Number,
      default: 100
    },
    // 带有默认值的对象
    propE: {
      type: Object,
      // 对象或数组默认值必须从一个工厂函数获取
      default: function () {
        return { message: 'hello' }
      }
    },
    // 自定义验证函数
    propF: {
      validator: function (value) {
        // 这个值必须匹配下列字符串中的一个
        return ['success', 'warning', 'danger'].indexOf(value) !== -1
      }
    }
  }

3. 子组件直接使用数据

<template>
  <div>
    {{title}}
    <button @click="open">测试</button>
  </div>
</template>

兄弟组件间传值

1.创建一个vue实例VueEvent.js

import Vue from 'vue'

var VueEvent=new Vue()

export default VueEvent

2.在兄弟组件中引入

import VueEvent from '**/VueEvent.js'

3.传值和接收

//brother_one.vue
//广播-传值
VueEvent.$emit('toMsg',this.msg)
//监听-接收
VueEvent.$on('toMsg',(data)=>{console.log(data)})

判断类型的方法

typeof // 判断一个值属于哪种基本类型。其中特殊的有null==>object,function==>function
instanceof //构造函数的 prototype 是否出现在被检测对象的原型链上

Array.from

​ 将一个类数组对象或者可遍历对象转换成一个真正的数组。最基本的要求是有length,按数组下表排列好,没有的地方会填充undefined

​ 举例：

let obj={0:1,1:2,2:3,3:4,length:4}
// 将这个类数组转成==》[ {No:1,id:1},... ]
console.log(
	// 大多数数组的方法都可遍历，以及改变this的奇效  第三个参数用于拓展
    Array.from(obj,(item,key)=>{
        return {No:this.prefix+item,order:key}
    },{prefix:"No."})

)

new Object

let obj={0:1}
var newObj=new Object(obj)
console.log(newObj)
console.log(newObj===obj) //true 引用类型

let arr=[1,2,3]
var newArr=new Array(arr)
console.log(newArr==arr) //false

原始类型都是继承来的,方法来自于他的原型

Object.keys / Object.getOwnPropertyNames

抽取对象的键值

Object.keys抽取可枚举属性

Object.getOwnPropertyNames 忽略属性描述符(descriptor)

属性描述符(descriptor)

  • configurable: 是否可删除
  • enumerable: 是否可枚举(遍历)
  • value: 属性值
  • writable: 是否可修改

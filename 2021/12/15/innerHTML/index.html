innerHTML

2021-12-15 前端 315 字
html

区别：

  1. innerHTML:在控件中加html代码 ,就是设置一个元素里面的HTML,标签对文本信息有效。
  2. innerTEXT:打印标签之间的纯文本信息，显示标签，标签无效，低版本的火狐浏览器不支持。
  3. value:显示双引号中的全部内容，显示标签，标签无效。

innerHTML

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <script>
    function show() {
      var users = [{
          uid: 1,
          uname: "dingding",
          age: 18
        },
        {
          uid: 2,
          uname: "fangfang",
          age: 18
        },
        {
          uid: 3,
          uname: "wangwang",
          age: 18
        }
      ]
      //获取div对象
      var obj = document.getElementById("d1");
      strHTML=`
      <table border=1>
        <tr>
          <th>编号</th>
          <th>用户名</th>
          <th>年龄</th>
        </tr>`;
        for(var i=0;i<users.length;i++){
        strHTML+=`
        <tr>
          <td>`+`${users[i].uid}`+`</td>
          <td>`+`${users[i].uname}`+`</td>
          <td>`+`${users[i].age}`+`</td>
        </tr>`
        }
        strHTML+=`</table>`
      obj.innerHTML = strHTML;
    }
  </script>
</head>

<body>
  <div id="d1"></div>
  <button onclick="show()">显示</button>
</body>
</html>
